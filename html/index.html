Tree Decomposition in your browser

Paste a DIMACS CNF below and click Run. The primal graph is built, FlowCutter produces a tree decomposition, and the resulting decomposition is rendered below as an interactive graph.
Built on top of the original FlowCutter library by Ben Strasser, with graph libraries by Tuukka Korhonen and Matti Jarvisalo, and additional graph code by Kenji Hashimoto. Packaged as a library and browser UI by Mate Soos. All bugs belong to Mate Soos (soos.mate@gmail.com).
